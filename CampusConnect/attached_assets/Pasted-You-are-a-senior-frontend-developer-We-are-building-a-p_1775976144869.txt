3. ROUTING SYSTEM
========================
Routes:
- /login          → PublicRoute (redirect to /products if logged in)
- /register       → PublicRoute (redirect to /products if logged in)
- /products       → ProtectedRoute (redirect to /login if not logged in)
- /               → redirect to /products

========================
4. API SETUP
========================
- Axios instance with baseURL = http://localhost:5000/api/v1
- Request interceptor → attach token from localStorage
- Response interceptor:
  - If 401 → clear token → redirect to /login
  - Extract data from response.data.data automatically

========================
5. AUTH CONTEXT
========================
Store:
- user object
- token
- isAuthenticated (boolean)
- isLoading (boolean — for checking token on app load)

Functions:
- login(email, password) → calls API → stores token → sets user
- logout() → clears token → resets state
- On app load → if token exists → fetch /profile → restore user state
